# 📣 Промт для Cowork + копилка «Почему стоит работать в нашем приложении»

**Назначение файла (для нас, не для Cowork).**
Живая копилка обоснований — почему нашим приложением стоит пользоваться — по четырём
аудиториям: **ученик / учитель / директор школы / родители**. Пополняется по мере
развития системы (как `IDEAS.md` для идей), чтобы ничего не забыть и потом донести до
каждого. Внизу — **готовый промт для Cowork**: он берёт фактуру из этого файла и собирает
двуязычный (RU/DE) продающий документ. Правило: в промт добавляем только то, что **реально
работает в коде** (без обещаний вперёд).

## 🎯 Аргументы по аудиториям (черновик — дополняем)

### 🎓 Ученику — «почему тут учить язык легче»
- Учишься со своего телефона, **даже без интернета** (см. офлайн-ядро ниже).
- Всё — **на твоём родном языке** (10 языков интерфейса и переводов), не только на русском.
- Слово показывается с картинкой, озвучкой и примером; повторения приходят сами, когда пора
  (умный алгоритм SM-2 — не зубрёжка списком).
- Можно **говорить вслух** и приложение слышит и поправляет (произношение + AI-тренер Pablo).

### 👩‍🏫 Учителю — «экономит урок и даёт контроль»
- **Сфоткал страницу учебника/тетради → готовый интерактивный урок** (слова, переводы,
  картинки, 7 типов упражнений) собирается сам, за минуты.
- Видит **аналитику класса**: кто активен, точность, трудные слова, где застревают.
- **«Игра класса»**: раздаёт ученикам персональные фразы из слов урока — закрепление вслух.
- Печатные листы урока (A4) для работы без телефонов.

### 🏫 Директору школы — «почему школе выгодно (и не отказаться)»
- Один инструмент закрывает: подготовку урока, домашку, повторение, отчётность, вовлечение.
- **Мультиарендность** (Школа→Класс→Учитель→Ученик), каталог учебников, лимиты/тарифы.
- Работает офлайн → уроки не срываются из-за слабого интернета в классе.

### 👨‍👩‍👧 Родителям — «ребёнок реально занимается»
- Прозрачный прогресс, тёплый тон, раздел «Любовь к детям» (добрые фразы родитель↔ребёнок).
- Занятия идут и в дороге/без сети.

*(Разделы живые — Павел накидывает аргументы голосом, ассистент оформляет.)*

## 📚 ФАКТУРА (что приложение реально умеет — из кода)

Собрано по `docs/teacher-facts.md` (инвентаризация репозитория, каждый факт подтверждён кодом).

**Импорт урока по фото.** Учитель фотографирует страницу учебника и/или тетрадь-доску →
GPT-4o Vision извлекает слова, консолидирует (артикли, регистр, инфинитив, без дублей),
переводит на активные локали, рисует детские ИИ-картинки к словам без фото и генерирует
упражнения. Учебник и тетрадь обрабатываются раздельно, тетрадь сверяется с учебником, чтобы не
дублировать. Название/описание урока ИИ придумывает сам, если не задано.

**7 типов упражнений** (5 генерируются на каждое слово сразу + диктант и произношение отдельно):
1. **Флеш-карта** — слово озвучивается, клик открывает перевод, ученик сам оценивает память
   (не помню / с трудом / помню) → идёт в SM-2.
2. **Заполни пропуск** — предложение с `___`, ввод или выбор из вариантов; показывается перевод
   всего предложения на язык ученика.
3. **Выбери ответ** — слово изучаемого языка + 4 варианта перевода.
4. **Напиши предложение** — свободный ввод со словом; ИИ оценивает 0–5, даёт фидбек и исправление.
5. **Добавь букву** — слово с замаскированными буквами, первая всегда видна.
6. **Диктант** — слово озвучивается, показан только перевод; ученик пишет услышанное на слух.
7. **Проговори слова (произношение)** — микрофон + распознавание речи, оценка по схожести,
   фонетические подсказки и разметка ударения.

**Умные повторения (SM-2/SRS).** Очередь «на сегодня» собирается по алгоритму интервального
повторения; порядок внутри дня — случайный, диктант всегда последним.

**10 языков.** Интерфейс и весь контент (слова, упражнения, заголовки, разговорник) — на языке
каждого ученика; поддержка нескольких изучаемых языков (мульти-таргет).

**AI-тренер Pablo** — разговорная речь: текст и голос, распознавание речи, видео-аватар,
память диалога, режим слов урока, реакции «верно/неверно». Голосовой режим «как в Gemini».

**Аналитика класса для учителя** — сколько учеников и кто активен за 7 дней, общая точность,
по каждому ученику точность/слова/последний визит, трудные слова, точность по типам упражнений.

**«Игра класса»** — из слов урока генерируются персональные фразы (вопрос-ответ), переводятся
на 10 локалей, раздаются ученикам по кругу с push-уведомлением; экран ученика (фраза + перевод +
🔊 + «прочитал» + «в разговорник»), экран учителя (все фразы по ученикам). *(Реализован MVP:
асинхронная раздача; live-ведение хода — в планах, в проде пока нет.)*

**Ещё в системе:** Читалка (режимы Читать/Двуязычный/Разговор, перевод по фото-камере),
Разговорник, Грамматика-справочник, раздел «Любовь к детям», каталог репетиторов/школ с картой,
каталог учебников + PDF-импорт, личный словарь ученика (фото тетради → свои слова),
мультиарендность Школа→Класс→Учитель→Ученик, супер-админка с лимитами/тарифами.

## 🆕 Мега-достижение: полноценная работа офлайн (задеплоено)

**С интернетом (незаметно для ученика):** после входа приложение фоном скачивает весь словарь и
все упражнения пяти типов вместе с личным SRS-прогрессом в локальную базу телефона (IndexedDB),
плюс прогревает кэш маленьких картинок слов. Обновляется не чаще раза в 30 минут — трафик щадим.

**Без интернета:**
- 📚 **Словарь** — открывается полностью из локальной базы.
- ✏️ **Упражнения** — флеш-карты, «вставь буквы», «выбор ответа», «вставь слово», диктант;
  очередь «на сегодня» собирает локальный SM-2 — даже завтра без сети будут правильные повторения.
- Ответы копятся в локальной очереди; внизу экрана — жёлтая плашка «Офлайн — прогресс отправится
  при появлении сети» (на языке ученика, все 10 локалей).

**Когда сеть вернулась:** очередь ответов автоматически улетает на сервер и применяется той же
SM-2 логикой (при двух устройствах побеждает последний ответ по времени). Ученик ничего не нажимает.

**Честно НЕ работает офлайн (by design):** «Напиши предложение» и «Произношение» (нужен ИИ),
тренер, чат, переводы, загрузка фото.

**Как проверить на телефоне:**
1. Открой приложение с интернетом, подожди ~10 секунд (предзагрузка идёт фоном).
2. Включи авиарежим ✈️.
3. Открой Словарь и любое упражнение — всё работает, внизу жёлтая плашка.
4. Ответь на пару заданий, выключи авиарежим — прогресс долетит на сервер сам.
