## Контекст
- Стек: Fastify 4 (raw SQL) + React 18/Vite + PostgreSQL 16 + Docker Compose.
- Бэклог — **`IDEAS.md`** в корне репо: сверяйся, отмечай `[x]` с датой.
- Деплой сам: `git push` → `ssh gcloud-seosite` → `cd /home/seosite/translate && git pull` → `docker compose -f docker-compose.prod.yml build frontend backend && up -d`. БД: сервис `db`, юзер `german_app`, база `german_learning`.
- ⚠️ nginx-шлюз `seoshkin_nginx` НЕ трогать через `up` с volume-правками — только `nginx -s reload`.
- **Всё сразу на 10 локалей** (файлы i18n — `frontend/src/i18n/` или `frontend/src/store/i18n.js`, найди структуру `t.*`). Комментарии — русский, имена — английский.
- Правило проекта: **AiTrainer.jsx не трогать** без явной просьбы.
- Бери задачи по одной, собирай локально (`cd frontend && npx vite build`), деплой пачками, проверяй сайт 200.
