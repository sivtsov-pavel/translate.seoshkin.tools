- Стек: Fastify 4 + React 18/Vite + PostgreSQL 16 + Docker Compose. Деплой сам: `git push` → `ssh gcloud-seosite` → `cd /home/seosite/translate && git pull` → `docker compose -f docker-compose.prod.yml build frontend && up -d`.
- ⚠️ nginx `seoshkin_nginx` НЕ трогать. AiTrainer.jsx не трогать.
- i18n: **10 языков** — файлы `frontend/src/i18n/{ru,en,de,uk,es,fr,bg,tr,ar,sq}.js`, каждый экспортирует `export default {...}` с секциями `nav`, `dashboard`, `exercise`, `auth` и др. Доступ в компонентах: `const { t } = useI18nStore()` (`store/i18n.js`) → `t.<секция>.<ключ>`.
- Стиль v3 уже внедрён (токены `--blue/--gold/--ink/--surface`, lucide-иконки). Держись стиля.

## Задача — локализовать страницы Профиля и Настроек на 10 языков
**Проблема (Павел):** в разделе Профиль/Настройки почти всё на русском, не подчиняется локали интерфейса.

**Файлы:** `frontend/src/pages/Settings.jsx` (вкладки + вкладка «Настройки»/«Интеграции»/«Напоминания») и `frontend/src/pages/Profile.jsx` (вкладка «Профиль»: имя/аватар/email/смена пароля).

**Уже сделано (НЕ дублируй):** вкладки (`t.nav.tabProfile/tabSettings/tabIntegrations/tabReminders`), `t.nav.save/saveSettings/morningReminder/eveningReminder/milestones` уже добавлены в i18n. Кнопка «Включить уведомления»/«Тест push себе» — можно локализовать.

**Что сделать:**
1. Пройди по `Settings.jsx` и `Profile.jsx`, найди ВСЕ русские строки в JSX (заголовки секций, подписи полей, кнопки, плейсхолдеры, тосты «Сохранено»/«Ошибка», подписи шрифтов/размеров/цветов/тем, тексты про голос, интеграции, напоминания).
2. Для каждой добавь ключ в секцию (предлагаю новую секцию **`settings`** в каждом из 10 i18n-файлов; либо переиспользуй `nav`). Переведи на все 10 языков (ru/en/de/uk/es/fr/bg/tr/ar/sq). Арабский — RTL, но строка обычная.
3. Замени хардкод на `t.settings.<ключ>` (с фолбэком `|| 'русский'` на всякий).
4. Названия шрифтов (Roboto/Georgia/…) НЕ переводи — это имена; переводи только описания-сэмплы («Классический с засечками» и т.п.).
5. Собери локально (`cd frontend && npx vite build`), проверь что каждый i18n-файл парсится, задеплой, проверь сайт 200. Переключи язык интерфейса (аватар → «Ваш язык») и убедись, что Профиль/Настройки на выбранном языке.

**Совет по объёму:** строк ~46+ в Settings + поля Profile. Можно скриптом-node вставлять ключи в 10 файлов пачкой (после `nav: {` или новой секцией), как это уже делалось в проекте.
